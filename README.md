## 🚀 How to Use

1. **Copy** the desired `.liquid` file into your Shopify theme's `/sections/` folder.
2. **Add** the section via the Shopify Theme Editor (`Customize` → `Add Section`).
3. **Configure** all schema settings directly in the editor — no code editing required.

---

## 📁 Adding Preview Images

To add real screenshots to the collapsible previews:

1. Create a `/previews/` folder in this repository.
2. Add a screenshot named after each file (e.g., `about-story-section.png`).
3. Replace the placeholder text in each `<details>` block with:

